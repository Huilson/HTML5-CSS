TERMOS

Para escrever em negrito usa se STRONG.

Para escreve em itálico usa se EM.

Para marcar um texto use se MARK.

Para alterar o tamanho de texto usa se SMALL

Para riscar um texto usa se DEL.

Para sublinhar um texto usa se INS.

Para fazer algo sobrescrito usa se SUP

Para fazer algo subrescrito usa se SUB

Para escrever de forma monoespaçada, especialmente usada para códigos, usa se CODE.Exemplo:


            int main(){
                int num = 1;
                scanf(&num);
                if(num % 2){

                }
                printf("Fim");
            }
